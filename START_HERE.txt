🚀 Quick Start (choose a method):
═══════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────┐
│ 1️⃣  Local Testing (30 seconds) - For development                │
└─────────────────────────────────────────────────────────────────┘

   Just open: index.html

   ✅ App will work immediately in Demo mode
   ✅ All features available
   ✅ Complete demo data

┌─────────────────────────────────────────────────────────────────┐
│ 2️⃣  Deploy to GitHub Pages (5 minutes) - Free HTTPS             │
└─────────────────────────────────────────────────────────────────┘

   Terminal:
   
   git init
   git add .
   git commit -m "TG Data Inspector"
   git remote add origin https://github.com/username/tg-inspector.git
   git push -u origin main

   GitHub:
   
   Settings → Pages → Source: main → Save
   
   URL: https://username.github.io/tg-inspector/

┌─────────────────────────────────────────────────────────────────┐
│ 3️⃣  Connect to Telegram Bot (2 minutes)                         │
└─────────────────────────────────────────────────────────────────┘

   Telegram:
   
   1. @BotFather → /mybots → Your Bot
   2. Bot Settings → Menu Button
   3. Enter URL: https://username.github.io/tg-inspector/
   
   ✅ Open bot on Telegram → Menu Button → App works!

┌─────────────────────────────────────────────────────────────────┐
│ 4️⃣  Quick Deploy with Vercel (1 minute) - Fastest               │
└─────────────────────────────────────────────────────────────────┘

   Terminal:
   
   npx vercel
   
   ✅ Gives you URL immediately: https://project.vercel.app
